## ✍ 나의 풀이 흐름

```javascript
1번째
function solution(arr){
    let min = arr.reduce((acc, cur) => acc < cur ? acc : cur)
    return arr.length == 1 ? [-1] : arr.filter( v => v !== min)
}
(acc, cur) => acc < cur ? acc : cur; -> (acc, cur) => Math.min(acc, cur)
```
